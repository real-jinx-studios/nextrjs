Registering EZConvert

Once you've installed EZConvert and went through the license activation, you can verify if all the features have been successfully registered by starting EZConvert and checking the Help/About menu.


EZConvert About Dialog

You can find some general information about your computer in the System Information section.

Pressing the Details button will open additional window with information about the registered features, software's expire date and support availability as described by the above picture.

Site License Registration

With the release of EZConvert 6, the Professional edition of the software now comes with a Site License included in the package. The Site License will activate after pasting special license file issued by us into EZConvert's installation folder.

As long as the Site License file is present inside the installation folder EZConvert's GUI, Command Line and Watch Folder can be used for conversions.

The Site License can be loaded from any location via the /sitelic parameter:

/sitelic:"\\Full UNC Path\EZConvert6.lic".  

The /sitelic paramater can be applied to:

▪EZConvert's GUI by adding it to the Target field of the EZConver't shortcut

▪Command Line by adding it to the command line directly

▪Watch Folder Service's configuration with the help of the SC command.

Please contact us at support@eztitles.com for more details.
