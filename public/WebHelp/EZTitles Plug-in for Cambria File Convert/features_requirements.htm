Features and Requirements

System Requirements

The EZTitles Plug-in doesn't require any additional software or hardware. It will run smoothly with every computer which already has Cambria File Convert or Cambria Manager installed. Any workstation which meets Cambria's system requirements will be able to use the EZTitles Plug-in as well.

Supported File Formats

The EZTitles Plug-in for Cambria can load subtitles from the following files:  

▪EZT  and EZTXML

▪EBU subtitle files  (.STL)

▪Compressed PAC subtitle files (.PAC)

▪Unicode PAC subtitle files (.FPC)

▪Win2020 text subtitle files (.TXT)

▪890 subtitle files (.890)

▪Videotron Lambda Cap (.cap) files

▪WebVTT (.vtt) subtitle files

▪SubRip (.srt) subtitle files

▪MicroDVD (.sub) subtitle files

▪Windows Media Player SAMI subtitle files (.SMI, .SAMI)

▪Plain ASCII with timecode subtitle files (.TXT)

▪RTF subtitle files (.RTF)

▪Excel XLS and XLSX Workbook subtitle files (.XLS)

▪QuickTime Text subtitle files (.TXT)

▪DLP Cinema™ XML subtitle files (.XML)

▪DCDM (Digital Cinema Distribution Master)SMPTE 428-7-2007 XML Subtitles

▪DAS subtitle files (.DAS)

▪Softitler subtitle files (.TXT)

▪Avid DS Nitris subtitle files (.TXT)

▪Timed Text XML (DFXP)- TTML ver.1 and TTML ver.2  subtitle files (.XML)

▪Ooyala Timed Text XML

▪IMSC1 Timed Text XML

▪IMSC1.1 Timed Text XML

▪Netflix Timed Text (NFLX-TT)

▪Disney+ Japanese IMSC 1.1

▪EBU Timed Text (EBU-TT and EBU-TT-D)

▪EBU-TT and EBU-TT-D for BBC

▪Apple iTunes Timed Text (.itt)

▪SMPTE-TT Subtitles (.xml)

▪Substation Alpha subtitle files (.SSA/.ASS)

▪Apple DVD Studio Pro text script (.stl)

▪Spruce DVD Maestro STL Text Script

▪Final Cut Pro XML Interchange Format

▪Adobe Encore DVD text script (.txt)

▪ Sony DVD Architect Script

▪Roxio DVDIt Pro Subtitle Script

▪Swift Interchange Format (.sif)

▪Universal Subtitle Format (.usf)

▪Inscriber SG (.txt)

▪OVR subtitle files (.OVR)

The Plug-in also accepts Closed Captions specific files:

▪Scenarist Closed Caption Format (.SCC)

▪SMPTE-TT Closed Captions (.xml)

▪Crackle 1.1 compliant SMPTE-TT

▪CFF-TT

▪Cheetah CAP (.CAP)

▪MCC CEA-608/708 captions (.mcc)

▪CPC-715 Online Caption Format (.onl)

▪Captions Inc. Files (.cin)

▪Cheetah ASC (.ASC)

▪NCI CAP (.CAP)

▪Ultech Captions (.ULT)

▪TDS Captions (.TDS)

▪ECF Captions for theater projection (.ECF)
