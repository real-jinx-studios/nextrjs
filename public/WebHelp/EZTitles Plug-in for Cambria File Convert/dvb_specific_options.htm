DVB Specific Options

The DVB subtitle stream has several other properties which can be modified from the respective section:

 

Pixel color depth

Determines the quality of the DVB subtitles. Choosing greater value will increase the quality of the DVB subtitles but it will also increase their size too.

Composition page ID

Enables you to specify Id for the composition page

Subtitling type

Normal subtitles or subtitles that have descriptive function for the deaf and hard of hearing.

Language code

Three letters language code specified by the ISO 639-2. The DVB standard allows different languages to be transmitted simultaneously and viewers are able to choose which one to activate by the subtitle selection menu of their TV set.

PID

Enables you to specify ID of the stream.

Erase incoming DVB data

Erases all DVB data already present in the video clip.

Note: When using multiple filters every filter except the first one should have this options turned off. For usage of multiple Plug-in presets please refer to the following topic.

Skip Display Definition Segment

Discards Display Definition Segment from the DVB stream. By default this options is turned off.

Use Region Fill Flag

Forces the region_fill_flag. By default this option is turned off.

Use 0x00000000 for transparent CLUT entry

The transparent color could be 0x00000000 or black with maximum possible level of transparency (255). By default this option is turned on.

Full width regions

Instead of creating display regions with the size of the text the plugin creates a region with the width of the video instead. By default this options is turned off.

Full width bitmaps

Creates bitmaps to fill in the whole display region. Some decoding devices might not be able to correctly display partially filled or empty regions. By default this options is turned off.

Send non-empty PCS on hide subtitle

By default each subtitle will hide by transmitting an empty page which doesn't contain any regions. This, however, might turn into a problem for some devices and by activating this option EZTitles will create a page containing region that equals the size of the lastly displayed one, instead of a page without a region.

Send empty bitmap

The displayed region for the non-empty PCS will be filled with transparent bitmap. By default this option is off.
