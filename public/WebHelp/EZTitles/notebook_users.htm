Notes for the notebook users

As mentioned before some commands in EZTitles use keys located on the numeric keypad.

These keys are not accessible on the notebooks without numeric keyboards.

Because of that some of the most important shortcuts are duplicated.

If the shortcuts are not suitable for you or are located on an inconvenient place on the keyboard, you can easily change them using Tools/Customize... menu.

Timecode Shortcuts:

In-cue

Alt+]

Sets In-cue value from the current time-code

Out-cue

Alt+P

Sets Out-cue value from the current time-code

Out/In-cue

Alt+[

Sets current subtitle Out-cue and next subtitle In-cue. The command can operate in the following modes:

- capture current subtitle Out-cue and calculate the next In-cue based on min. interval;

- capture next subtitle In-cue and calculate the current subtitle Out-cue min. interval frames backward.

The command behaviour is customized in Preferences dialog.

In-cue Override

Alt+Ctrl+]

Sets In-cue value from the current time-code. Preview mode only.

Out-cue Override

Alt+Ctrl+P

Sets Out-cue value from the current time-code. Preview mode only.

Out/In-cue Override

Alt+Ctrl+[

Sets current subtitle Out-cue and next subtitle In-cue. Preview mode only.

In-cue + 1

Ctrl+'

Increases current In-cue value with a frame

In-cue - 1

Ctrl+;

Decreases current In-cue value with a frame

Out-cue + 1

Ctrl+/

Increases current Out-cue value with a frame

Out-cue - 1

Ctrl+.

Decreases current Out-cue value with a frame

In-cue + 1, Chained

Alt+Ctrl+'

Increases current in-cue with one frame, out-cue of the previous subtitle is also increased with one frame.

In-cue - 1, Chained

Alt+Ctrl+;

Decreases current in-cue with one frame, out-cue of the previous subtitle is also decreased with one frame.

Out-cue + 1, Chained

Alt+Ctrl+/

Increases current out-cue with one frame, the in-cue of the next subtitle is also increased with one frame

Out-cue - 1, Chained

Alt+Ctrl+.

Decreases current out-cue with one frame, the in-cue of the next subtitle is also decreased with one frame.

Move Rows Left/Right:

Move Row Left

Alt+J

Moves current row one character to the left

Move Row Right

Alt+K

Moves current row one character to the right

Move Subtitle Left

Alt+Ctrl+J

Moves current subtitle one character to the left

Move Subtitle Right

Alt+Ctrl+K

Moves current subtitle one character to the right
