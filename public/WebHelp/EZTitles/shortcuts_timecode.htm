Timecode Shortcuts

In-cue

Num Enter

Sets the In-cue’s value from the current time-code

Out-cue

Num Ins

Sets the Out-cue’s value from the current time-code

Out/In-cue

Num Del

Sets the current subtitle’s Out-cue and the next subtitle’s In-cue. The command can operate in the following modes:

- capture current subtitle’s Out-cue and calculate the next In-cue based on min. interval;

- capture next subtitle’s In-cue and calculate the current subtitle’s Out-cue min. interval frames backward;

- capture next caption’s In-Cue (Usable in Closed Captions mode only).

The command’s behaviour is customized in Preferences window.

In-cue TC Override

Ctrl+Num Enter

Sets In-cue value from the current time-code. Preview mode only.

Out-cue TC Override

Ctrl+Num Ins

Sets Out-cue value from the current time-code. Preview mode only.

Out/In-cue TC Override

Ctrl+Num Del

Sets current subtitle’s Out-cue and the next subtitle’s In-cue. Preview mode only.

In-cue
snap to shot override

Shift+Num Enter

Capture the current subtitle’s In-cue and override the "Snap to shot changes option".

Out-cue
snap to shot override

Shift+Num Ins

Capture the current subtitle’s Out-cue and override the "Snap to shot changes option".

Out/In-cue
snap to shot override

Shift+Num Del

Sets Out-cue’s value from the current time-code and the next subtitle’s In-cue at default interval and overrides the "Snap to shot changes option".

In-cue + 1

Num +

Increases current In-cue value with a frame

In-cue - 1

Num -

Decreases current In-cue value with a frame

Out-cue + 1

Shift+Num +

Increases current Out-cue value with a frame

Out-cue - 1

Shift+Num -

Decreases current Out-cue value with a frame

In-cue + 1, Chained

Alt+Num +

Increases current in-cue with one frame, out-cue of the previous subtitle is also increased with one frame.

In-cue - 1, Chained

Alt+Num -

Decreases current in-cue with one frame, out-cue of the previous subtitle is also decreased with one frame.

Out-cue + 1, Chained

Alt+Shift+Num +

Increases current out-cue with one frame, the in-cue of the next subtitle is also increased with one frame

Out-cue - 1, Chained

Alt+Shift+Num -

Decreases current out-cue with one frame, the in-cue of the next subtitle is also decreased with one frame.

Reset In-cue

Alt+Ctrl+Num Enter

Sets In-cue value to be at minimum interval from the previous Out-cue

Reset Out-cue

Alt+Ctrl+Num Ins

Sets Out-cue value to be at minimum interval from the next In-cue

Recalc In-cue

Shift+Num *

Recalculates In-cue value based on the current Out-cue, text length and reading speed

Recalc Out-cue

Num *

Recalculates Out-cue value based on the current In-cue, text length and reading speed

Fit Previous Out-cue

Num /

Moves the previous Out-cue value to the next In-cue to be at the minimum interval

Fit Current In-Cue

 

Fits current subtitle's In-cue to previous subtitle's Out-cue

Fit Current Out-Cue

 

Fits current subtitle's Out-cue to next subtitle's In-Cue

Fit Next In-Cue

Shift+Num /

Moves the next In-cue value to the current Out-cue to be at the minimum interval
