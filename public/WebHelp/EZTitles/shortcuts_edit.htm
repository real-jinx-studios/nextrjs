Edit and Format Shortcuts

Word Up

Ctrl+Up

Moves the first word of the current row onto the previous row

Word Down

Ctrl+Down

Moves the last word of the current row onto the next row

Delete Whole Word

Ctrl+Del

Deletes the current word

Delete Word to the Left

Ctrl+BkSp

Delete Word to the Left

Move Subtitle Rows Up

Alt+Ctrl+Up

Moves the whole subtitle up one row

Move Subtitle Rows Down

Alt+Ctrl+Down

Moves the whole subtitle down one row

Fine Move Subtitle Up

Alt + Shift + Up

Moves the whole subtitle up with the vertical offset configured from the Vertical Text options menu.

Fine Move Subtitle Down

Alt + Shift + Down

Moves the whole subtitle down with the vertical offset configured from the Vertical Text options menu.

Row Clear

Alt+Space

Clears current subtitle row contents

Row Remove

Alt+Del

Removes current subtitle row

Row Clear Rest

Shift+Del

Clears current row from the current position to the rest of the row

Move Row Left

Alt+Num Left

Moves current row one character to the left

Move Row Right

Alt+Num Right

Moves current row one character to the right

Move Subtitle Left

Alt+Ctrl+Num Left

Moves current subtitle one character to the left

Move Subtitle Right

Alt+Ctrl+Num Right

Moves current subtitle one character to the right

Box subtitle

Alt+Ctrl+B

Create a box around the current subtitle

UPPERCASE selected

Ctrl+U

UPPERCASE the selected text

UPPERCASE row

Alt+Ctrl+U

UPPERCASE the current subtitle row

lowercase selected

Ctrl+B

lowercase the selected text

Add to Drafts

Ctrl+L

Adds current subtitle to the Draft list

Empty

Alt+Ctrl+Space

Clears the whole subtitle

Remove

Alt+Ctrl+BkSp

Removes the current subtitle

Insert

Alt+Ctrl+Ins

Inserts a new blank subtitle

Insert before

Alt+Shift+Ins

Inserts a new subtitle before the current one

Split

Alt+Ctrl+Enter

Splits the current subtitle at the cursor position

Merge with Next

Alt+Ctrl+=

Merges current subtitle text with the next subtitle text

Move Text to Next

Alt+Ctrl+-

Moves text from the cursor position onwards to the next subtitle

Move Text to Previous

Alt+Shift+-

Moves the text before the cursor to the previous subtitle

Undo

Ctrl+Z

Reverses the last command or deletes the last entry you typed

Group Undo

Alt+Ctrl+Z

Restores the contents of the last edited subtitle

Redo

Ctrl+Y

Reverses the action of Undo command

Select All

Ctrl+A

Selects all text in the current subtitle

Cut

Ctrl+X

Removes the selection from the current subtitle and places it on the Clipboard

Copy

Ctrl+C

Copies the selection from the current subtitle on the Clipboard

Paste

Ctrl+V

Inserts the contents of the Clipboard at the current cursor position, and replaces any selection

Paste without formatting

 

Discards any formatting (colors, italics, background colors, etc) styles and pastes the text plain from the clipboard.

Clear

Del

Deletes the selected text without inserting it on the Clipboard

Italic

Ctrl+I

Puts all subsequent text into italic or switch it off

Row Italic

Alt+Ctrl+I

Puts whole row into italic or switch it off

Subtitle Italic

 

Puts the whole subtitle in italics.

Double Height

Ctrl+D

Switch on/off the Double Height (Teletext mode only)

Row Double Height

Alt+Ctrl+D

Puts whole row into double height or switch it off (Teletext mode only)

Find...

Ctrl+F

Searches for specified text into current subtitles file

Replace...

Ctrl+R

Searches for and replaces the specified text into current subtitles file

Goto...

Ctrl+G

Moves to the specified subtitle by number or timecode
