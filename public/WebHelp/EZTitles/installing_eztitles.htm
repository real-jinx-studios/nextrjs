Installing EZTitles

Installing EZTitles is pretty easy. You just have to follow the simple instructions on each page of the wizard and click the Next button.

So we will focus only on the pages where you will be asked to make a decision.

After you have read and agreed with the EZTitles software license agreement, you will see the following dialog:

Here you need to specify the directory, where EZTitles will be installed in. Your Program Files folder is selected by default, but you can change it, if you prefer. If you’re unsure, just click on the Next button.


Start Menu folder selection

Afterwards you’ll be asked to specify the Start Menu folder where the program will create and place it’s shortcuts. If you want to create a subfolder for the program you can use the “\” for a separator as in

“EZTitles Development Studio Ltd\EZTitles 6”.

You can also skip this folder creation by checking the Don’t create a Start Menu folder checkbox.

 

From the next dialog you can choose if you want to make EZTitles the default program for PAC, 890, EBU(.stl), CAP, CIN, SCC files.

If selected, when you double click on a file of such type in Explorer, EZTitles is started and the file is automatically loaded.

You can also add additional shortcuts for starting EZTitles from the Desktop and from the Quick Launch toolbar. Of course, you don’t have to create any shortcuts at all if you don't want to.


Additional options

EZTitles uses the integrated spell checker with Microsoft Word, OpenOffice,  Libre Office  or the Windows built-in spell checking options:


Spell Checking options

Disabling the spell checker may solve problems causing EZTitles to stop responding during start-up because of a damaged or not properly installed Office package.

The Windows built-in option is available on computers running Windows 8, Windows 8.1 or Windows 10. The spell checking API is unfortunately not available on Windows 7.

Later on you can enable the spell checker again using Tools/Proofing ... menu.

 

At the last dialog you will be asked to import the keyboard shortcuts, Zero Subtitle Templates and Project Templates from your previously installed version of EZTitles if you have any.

If you don’t want to use the old shortcuts or you are running EZTitles on this computer for the first time ever, don’t use this option.


Settings migration

EZTitles will ask you to migrate your settings from previous installation only when you're installing EZTitles ver. 5 for the first time.
