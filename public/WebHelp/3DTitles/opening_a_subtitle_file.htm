Opening a subtitle file

To start your project you must load a subtitle file into 3DTitles. Our software supports the native for EZTitles, .ezt, file format only.

You can easily do that by selecting Open... from the File menu or by using the keyboard shortcut - Ctrl+O.

You can either select a subtitle file from a list of your recent projects by choosing Open recent from the File menu.

If you open your file that way its corresponding video will be automatically loaded in 3DTitles.

Furthermore, 3DTitles remembers the exact subtitle you have been at last time you have saved your project and will point right onto it. Thus you can continue your work right away without having to find the right subtitle first.
