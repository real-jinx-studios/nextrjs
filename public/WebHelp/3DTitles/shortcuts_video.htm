Video Navigation and Timecode Shortcuts

In-Cue + 1

Ctrl + -

Increase current subtitle's in-cue value with a frame

In-Cue - 1

Ctrl + =

Decrease current subtitle's in-cue value with a frame

Out-Cue + 1

Ctrl + [

Increase current subtitle's out-cue value with a frame

Out-Cue - 1

Ctrl + ]

Decrease current subtitle's out-cue value with a frame

Play

Alt+Right

Starts playing current video file

Pause

Alt+Down

Pauses the video playback

Play 1.5x

Ctrl+Alt+Right

Starts playing of the current video file with speed 1.5

Pause/Play

Pause / Space

Pauses/Resumes the video playback

Speed Up

Alt+Shift+Up

Increases the video playback speed x1.5, x2, x4 and etc

Speed Down

Alt+Shift+Down

Decreases the video playing speed

Fast Forward

Alt+Shift+Right

Starts playing of the current video stream with fixed fast speed

Fast Backward

Alt+Shift+Left

Starts reverse playing of the current video stream with fixed fast speed

Jump Back

Alt+BkSp

Jumps back 10 seconds and starts playing forward at normal speed

Jump Forward

Alt+Shift+BkSp

Jumps forward 10 seconds and starts playing at normal speed

Next Frame

Alt+Up

Pauses playing video and shows next frame

Previous Frame

Alt+Down

Pauses playing video and shows previous frame

First Frame

Alt+Home

Jumps to the first frame of current video file

Last Frame

Alt+End

Jumps to the last frame of current video file

Previous Shot Change

Alt+[

Goto previous shot change

Next Shot Change

Alt+]

Goto next shot change

Play Current Subtitle

F4

Plays video for current subtitle

Preview Subtitles Only

F5

Toggles preview of the subtitled scenes of the video
